# Atelier

Bem-vindo ao repositório do projeto **Atelier**. Este projeto foi inicializado utilizando React, TypeScript e Vite.

## 🚀 Tecnologias Utilizadas

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)

## 📦 Pré-requisitos

Antes de começar, certifique-se de ter o [Node.js](https://nodejs.org/) instalado na sua máquina.

## 🛠️ Como rodar o projeto

1. Acesse o diretório do projeto:
   ```bash
   cd Atelier
   ```

2. Instale as dependências:
   ```bash
   npm install
   # ou
   yarn install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   # ou
   yarn dev
   ```

4. O aplicativo estará disponível no navegador, geralmente no endereço `http://localhost:5173`.

## 🏗️ Build para Produção

Para criar a versão de produção otimizada:
```bash
npm run build
# ou
yarn build
```
Os arquivos compilados serão gerados dentro da pasta `dist/`.
